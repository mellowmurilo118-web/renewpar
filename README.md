# Renew Part Bank - Homepage Clone

A Vite + React + Tailwind CSS clone of the Renew Part Bank homepage.

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack
- **Vite** - Build tool & dev server
- **React 18** - UI library
- **Tailwind CSS 3** - Utility-first CSS framework

## Structure

```
src/
  components/
    Navbar.jsx      - Sticky top nav with dropdown menus
    Hero.jsx        - Full-screen hero section
    Support.jsx     - Support banner
    About.jsx       - About us section
    Services.jsx    - 4 service cards
    Gallery.jsx     - Portfolio gallery
    Clients.jsx     - Partner logos strip
    Values.jsx      - Mission & values
    WhyUs.jsx       - Vision & mission
    News.jsx        - Latest news cards
    Footer.jsx      - Footer with newsletter
  App.jsx
  main.jsx
  index.css
```
